# Knife-Throw
Учебный проект из курса Udemy

Knife Throw

2D Arcade Game | Godot Engine

Небольшой аркадный проект, разрабатываемый в рамках изучения игрового движка Godot.

Игрок метает ножи в вращающуюся цель.
Главная задача — попасть в мишень, не задев уже воткнутые ножи.
С каждым уровнем сложность возрастает за счёт изменения скорости и поведения цели.

Статус проекта

В активной разработке
Проект постепенно дорабатывается и расширяется.
